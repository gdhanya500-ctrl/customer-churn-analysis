# Telco Churn Project Submission

## Included deliverables
- Notebook (editable): `Telco_Churn_Analysis.ipynb`
- Notebook (executed): `Telco_Churn_Analysis.executed.ipynb`
- Presentation (9 slides): `Telco_Churn_Presentation.pptx`
- Full output artifacts: `outputs/` folder

## What this submission covers
- Data loading, cleaning, missing-value handling, data-type conversion
- Categorical encoding and train/test split
- Churn model training and performance evaluation
- Explainability with SHAP (plus ELI5 attempt)
- Top churn-driver identification
- SQL business analysis
- Refined segmentation: At Risk, Loyal, Dormant
- Segment-wise retention strategies and final recommendations

- `outputs/sql_segment_analysis_refined.csv`
